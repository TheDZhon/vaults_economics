# Code of Conduct

This project follows the Contributor Covenant, version 2.1.

We expect all participants to be respectful, inclusive, and constructive.
Harassment, discrimination, and abuse are not tolerated.

## Enforcement

Maintainers may remove, edit, or reject contributions that violate this code.

## Reporting

If you experience or witness unacceptable behavior, please report it via GitHub.
Use a private security advisory or contact the maintainer through GitHub.
